# Repo Knowledge Base Schema

This repository uses `.repo-kb/` to maintain durable development knowledge.

See `.agents/skills/repo-kb/references/schema.md` for the full schema when this repository vendors the skill. If the skill is not vendored, see the installed `repo-kb` skill's `references/schema.md`.

The upstream skill should be installed and updated through `gh skill`. Vendored copies can be refreshed from the installed skill with `repo_kb.py vendor --force`; local repository knowledge belongs here.

Local deviations should be documented in this file.
