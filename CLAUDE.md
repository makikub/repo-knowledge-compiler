# Project Instructions

This repository uses `.repo-kb/` as the source of truth for durable agent guidance.

## Always Follow

- Before substantial work, check `.repo-kb/index.md` and relevant `.repo-kb/pages/`.
- During code review, use `REVIEW.md` and relevant `.repo-kb/review-aspects/`.
- Propose new `.repo-kb` entries for repeated review findings, incidents, design decisions, and important repo-specific practices.
- Do not make large direct edits to generated guidance. Update `.repo-kb/`, then run compile.

See `.repo-kb/generated/claude-context.md`.
